Se va crea cate un folder pentru fiecare animal.
Numele folder-ului va fi numele animalului.
In folder se vor insera imaginile corespunzatoare animalului

EXEMPLU

folder: leopard

continut folder:
leopard1.jpeg
leopard2.jpeg 
leopard3.jpeg

folder: leu

continut folder:
leu1.jpeg
leu2.jpeg
leu3.jpeg
